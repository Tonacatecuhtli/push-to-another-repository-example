# Exmaple
```yaml
name: Pushes docs to one.devops
  uses: Tonacatecuhtli/github-action-push-to-another-repository@master
  with:
    source-directory: <the-folder-you-want-to-copy>
    destination-directory: <folder>/<in>/<dest>/<repo>
    destination-github-username: <your-username>
    destination-repository-name: <your-reponame>
    user-email: <your@workemail.net>
```
